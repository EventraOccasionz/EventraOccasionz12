# System Manuals & Audits — `/docs/`

## Purpose
This directory houses architecture manuals, system walkthroughs, database design schemas, and developer auditing guidelines for the EventraOccasionz platform.

## Ownership
- **Owner Module**: Core Documentation
- **Design Intent**: Static markdown documentation describing workflows and requirements.

## Guidelines
- **Allowed Files**: Markdown documentation, visual diagram configurations, text files.
- **Forbidden Files**: Executable code files or assets intended for runtime browser compilation.
